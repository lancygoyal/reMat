## reMat

Powered By React, Redux and Material.

### Includes

- React 16
- Redux
- Material-UI

### Running it

```
npm install
npm start
```

### Folder structure

    .
    ├── src
    │   ├── components          # Dumb components
    │   ├── config              # App configrations
    │   ├── constants
    │   ├── containers          # Smart components
    │   ├── redux               # Redux modules
    │   ├── utilities
    │   ├── App.js              # Root component
    │   ├── index.js
    └── package.json            # Node dependencies
